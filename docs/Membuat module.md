# Membuat module

- Untuk membuat module baru, caranya cukup mudah, kita bisa membuat class dengan
  annotation @Module atau disebut juga decorator dan tambahkan import di application module
- atau cara lebih mudah bisa menggunakan perintah juga guys :
  nest generate module nama path
- Misal kita akan module user di path utanam (src), sehingga tidak perlu
  menyebutkan path nya, kita bisa gunakan perintah :
  nest generate module user
